## 📌 Overview

Automates deployment, validation and execution of AsBuiltReport for Veeam Backup & Replication.

Includes:
- Environment validation
- Online/offline module handling
- PowerShell compatibility (5.1 / 7)
- Veeam connectivity
- Report generation

---

## 🚀 Usage

```powershell
.\vbr_asbuilt_v19.ps1
```

---

## 📂 Output

Default:
```
script_path\report
```

---

## ⚠️ Note

Veeam v13 is not supported by the official AsBuiltReport module.
